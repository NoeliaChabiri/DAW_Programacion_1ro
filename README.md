# DAW - Programación  / Programming 

Este repositorio está dedicado a las actividades que realizo en **Programación 1 (Java)** durante mi carrera.  
This repository is dedicated to the activities I complete in **Programming 1 (Java)** during my first year of study.

## Objetivo / Purpose

- Guardar y organizar mis proyectos y prácticas en Java.  
- Store and organize my projects and practice exercises in Java.

- Registrar mi progreso a lo largo del tiempo.  
- Track my progress over time.

- Servir como referencia personal para futuros proyectos y estudios.  
- Serve as a personal reference for future projects and coursework.

## Estructura del repositorio / Repository Structure

- **EjerciciosBasicos / BasicExercises:** Ejercicios iniciales de Java, bucles, condicionales y estructuras simples.  
- **EjerciciosIntermedios / IntermediateExercises:** Ejercicios más complejos, manejo de arrays, objetos y métodos.  
- **ProyectosPersonales / PersonalProjects:** Proyectos completos, experimentos y prácticas avanzadas.

> Nota: Cada carpeta puede contener subcarpetas por tema o ejercicio según lo necesites.  
> Note: Each folder may contain subfolders by topic or exercise as needed.

## Uso / Usage

- Cada archivo `.java` se puede abrir y ejecutar en tu entorno de desarrollo (IDE) favorito.  
- Each `.java` file can be opened and run in your favorite development environment (IDE).

- Los proyectos completos están organizados en carpetas y listos para compilar y ejecutar.  
- Complete projects are organized in folders and ready to compile and run.
